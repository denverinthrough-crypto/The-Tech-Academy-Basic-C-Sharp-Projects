# The Tech Academy – Basic C# Projects

## Introduction
This repository contains my coding projects completed as part of **The Tech Academy Basic C# course**. These projects demonstrate my understanding of fundamental programming concepts using C# and the .NET framework.

## What This Repository Includes
- Console applications
- C# fundamentals (variables, loops, conditionals)
- Object-Oriented Programming concepts
- Methods, classes, and namespaces
- Practice projects and assignments from the course

## Projects

###  TwentyOne Card Game
A console-based card game built in C#, demonstrating:
- Object-Oriented Programming (classes, inheritance, interfaces)
- Game logic and flow control


###  Car Insurance Quote App
An ASP.NET Core MVC web application that calculates car insurance quotes, demonstrating:
- Full CRUD operations with Entity Framework Core (Code-First)
- Quote calculation logic (age brackets, DUI and coverage multipliers)
- MVC architecture (Models, Views, Controllers)
- SQL Server database integration

###  Newsletter App
An ASP.NET Core MVC web application for managing newsletter subscriptions, demonstrating:
- Entity Framework Core Database-First scaffolding
- LINQ projections and queries
- Tag helpers and Razor Views
- SQL Server database integration

###  Student Database (Code-First)
A console application demonstrating Entity Framework Core Code-First workflow:
- Designing a database entirely from C# classes
- Auto-generating SQL tables from models
- Inserting records via EF Core

## Skills Demonstrated

- SQL Server (CRUD operations)
- Object-Oriented Programming (abstract classes, interfaces, inheritance, polymorphism)
- LINQ and lambda expressions
